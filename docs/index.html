About This Archive

Archive of question papers and exercise sheets from IISER Mohali courses. Collected by students over several years.

What's here

  • Course-wise folders containing past quizzes, minors, majors, endsems, and exercise sheets
  • Formats: PDF (preferred), images (PNG/JPG), office docs (DOCX), and the occasional ZIP
  • Materials from Biology, Chemistry, HSS, IDC, Math, and Physics departments

Contributing

Pull requests welcome! You can contribute via:

  • Web: Use our Contribution Portal to upload files easily
  • GitHub: Fork the repository on GitHub, upload files, and open a pull request
  • File structure: <SUBJECT>/<COURSE_CODE>/<YEAR>/<file.pdf>

Quality varies; some scans are rough. This is a student-run archive intended for learning and exam prep. Use at your own risk.
