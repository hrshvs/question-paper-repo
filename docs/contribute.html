Contribution Guidelines

  • Authentication: Sign in with your IISER Mohali email (@iisermohali.ac.in)
  • File formats: PDF (preferred), JPG, PNG, DOCX, PPTX, XLSX, ZIP
  • File size: Maximum 7.5 MB per file (Cloudflare limit). Upload multiple files in one go - they'll be batched automatically!
  • Organization: Select subject from dropdown, enter 3-digit course code and 4-digit year
  • File naming: You can edit filenames after selecting them by clicking on the filename
  • Quality: Ensure files are clear, readable, and properly named
  • Privacy: Remove personal information (names, roll numbers) before uploading
